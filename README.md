# Weather App
A simple and responsive weather application that allows users to check the current weather conditions for any city. Built with HTML, CSS, JavaScript, and the OpenWeatherMap API.
## Features
- **Real-time Weather Data**: Fetches current temperature, humidity, and wind speed.
- **City Search**: Users can search for weather data by entering a city name.
- **Error Handling**: Alerts users if a city is not found or if the API key is invalid.
- **Responsive Design**: Clean and simple interface.
## How to Use
1. Clone the repository or download the files.
2. Open [index.html](cci:7://file:///c:/Users/Manoswini/weather-app/index.html:0:0-0:0) in your web browser.
3. Enter a city name in the input field.
4. Click the "Search" button to view the weather details.
## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
